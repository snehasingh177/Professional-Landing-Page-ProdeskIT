# Professional-Landing-Page-ProdeskIT

A modern and responsive landing page for Prodesk IT, built using HTML, CSS, and JavaScript. Features a clean UI, responsive design, dark mode, glassmorphism effects, smooth animations, interactive navigation, and optimized performance across desktop, tablet, and mobile devices.

Technologies Used
HTML5
CSS3 (Flexbox & CSS Grid)
JavaScript (ES6)

Key Features
📱 Fully Responsive Design
🌙 Dark/Light Mode Toggle
✨ Glassmorphism UI
🎯 Interactive Navigation
🚀 Smooth Hover Animations
💻 Mobile-Friendly Layout
🎨 Modern & Clean User Interface
